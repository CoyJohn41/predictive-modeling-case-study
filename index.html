01 — Overview

Project Overview

This project applies the full data science pipeline to the Ames Housing dataset — a rich, real-world dataset with 79 features describing residential properties in Ames, Iowa. The goal is to predict sale prices using statistical reasoning and machine learning, while communicating findings clearly.

Unlike toy examples, this dataset requires genuine decisions at every stage: how to handle 19 variables with missing data, which features to engineer, which models to trust, and how to interpret predictions in terms a non-technical audience can understand.

05 — Insights

Key Insights

  • TotalSF and OverallQual together explain more variance than all other features combined — size and quality dominate housing price prediction.
  • Log-transforming the target variable is essential. Raw prices are right-skewed; models trained on log(price) generalize significantly better on expensive homes.
  • Lasso naturally zeroed out 34 of 79 features, providing an automatic feature selection mechanism with a clear statistical justification.
  • Gradient Boosting outperformed linear models by capturing nonlinear interactions — but Ridge regression within 15% of its performance with far greater interpretability.
  • Neighborhood is a stronger predictor than square footage alone — location encodes school quality, walkability, and demand that raw physical features cannot capture.
06 — Future Work

Future Improvements

  • Stack Ridge and Gradient Boosting predictions as a blended ensemble for lower variance.
